Cross Method Theory

Moment distribution for continuous beams
50
minutes

1 Why Cross Method?

The bridge is a continuous beam over 2+ supports. We need to find reactions at each building.

2 Fixed-End Moments

Start with fixed ends. Calculate moments as if joints can't rotate.

3 Distribution Factors

How stiffness determines moment distribution at each joint. Stiffer attracts more moment.

4 Carryover Factors

Half the moment carries to the far end. Iterate until balanced.

Manual Exercise

Solve a 2-span continuous beam
50
minutes

1 Problem Setup

2-span beam, uniform load, equal spans. Classic bridge scenario.

2 Calculate FEM

Fixed-end moments for each span. Use standard formulas.

3 Distribute & Carry

Apply Cross Method step by step. Show all work clearly.

4 Find Reactions

From final moments, calculate support reactions. These become foundation loads!

Robot Verification

Does our hand calculation match?
50
minutes

1 Model in Robot

Create the same 2-span beam. Define supports, sections, loads.

2 Run Analysis

Execute static analysis. View moment diagram.

3 Compare Results

Hand calculation vs Robot. Do moments match? Do reactions match?

4 Discuss Differences

If different, why? Round-off? Sign convention? Model error?
